# 👨‍💻 My Portfolio Website

A modern, responsive portfolio website showcasing my skills and projects with glassmorphism design.

## ✨ Features

- **Glassmorphism Design** - Modern glass-like UI effects
- **Responsive Layout** - Works on all devices (mobile, tablet, desktop)
- **Smooth Animations** - Interactive hover effects and transitions
- **Infinite Skills Scroll** - Dynamic skills showcase
- **Contact Form** - Animated floating labels
- **Social Links** - GitHub, LinkedIn, Facebook integration

## 🛠️ Technologies Used

- **HTML5** - Semantic markup
- **CSS3** - Advanced styling with glassmorphism effects
- **Font Awesome** - Icons
- **Google Fonts** - Baloo Bhai 2 typography

## 🎨 Design Features

- Custom CSS animations
- Backdrop blur effects
- Gradient borders
- Floating labels
- Responsive grid system
- Modern color scheme (#bdd1d6)

## 📱 Responsive Breakpoints

- Desktop: 1200px+
- Tablet: 768px - 1199px
- Mobile: 320px - 767px

## 🔧 Setup & Installation

1. Clone the repository:

   ```bash
   git clone https://github.com/your-username/portfolio-website.git
   ```

2. Open `index.html` in your browser

3. Customize content:
   - Update personal information in `index.html`
   - Replace profile image in `assets/profile.jpg`
   - Modify colors in CSS custom properties

## 📝 Customization

### Colors

Update the CSS custom properties in any CSS file:

```css
:root {
	--color: #bdd1d6; /* Primary color */
	--font-family: "Baloo Bhai 2", sans-serif;
}
```

### Content

- Update personal details in `index.html`
- Modify project descriptions
- Add your social media links
