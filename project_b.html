Project B: Task Management Dashboard

2023 Full Stack Developer Team of 3

Stage 1: Research & Planning

This project was born from the need for a lightweight, real-time task management solution for small teams. I researched existing tools like Trello and Jira, identified gaps, and drafted requirements for a simpler, more focused alternative. User stories and feature prioritization guided the initial planning.

My Role & Responsibilities:

  • Led requirements gathering and user research
  • Created feature prioritization matrix
  • Designed database schema and WebSocket architecture
  • Drafted initial sketches of the kanban board UI

Feature planning and user story mapping

Stage 2: UI/UX Design

Dashboard layout and kanban board design

The UI was designed around efficiency — drag-and-drop kanban columns, color-coded priority labels, and a clean sidebar for navigation. Charts for team productivity were integrated using Chart.js, providing visual analytics at a glance.

Design Highlights:

  • Drag-and-drop kanban board interface
  • Color-coded task priorities and status indicators
  • Interactive analytics dashboard with Chart.js
  • Dark mode support for reduced eye strain

Stage 3: Implementation

The backend utilized Django with Django Channels for WebSocket support, enabling real-time task updates across all connected clients. Next.js powered the frontend with server-side rendering and optimized state management. Redis was used as the channel layer for message brokering.

Technical Highlights:

  • Real-time updates via Django Channels (WebSocket)
  • Next.js with React state management
  • Redis message broker for channel layer
  • Team roles and permissions system
  • Email notifications for deadline reminders

WebSocket architecture for real-time collaboration

Stage 4: Results & Outcomes

The completed task management dashboard

The dashboard was tested with a team of 10 users over 4 weeks. Feedback was incorporated iteratively. The final product improved team productivity and provided valuable insights through analytics.

Results & Impact:

  • 30% improvement in team task completion rate
  • Real-time sync with under 100ms latency
  • Adopted by 3 student project teams
  • Positive feedback on drag-and-drop UX
