Project C: Portfolio & Blog Platform

2024 Solo Developer Independent Project

Stage 1: Ideation & Sketching

This independent project was created to build a personal brand presence online — combining a portfolio showcase with a blog CMS. I started by sketching page layouts on paper, defining the content structure, and planning the information architecture to ensure easy navigation for visitors.

My Role & Responsibilities:

  • Sole designer and developer of the entire platform
  • Created hand-drawn sketches and digital wireframes
  • Planned content hierarchy and SEO strategy
  • Defined database models for blog posts and portfolio items

Hand-drawn sketches and digital wireframes

Stage 2: Design System & UI

Color palette, typography, and component library

I created a cohesive design system with a defined color palette, typography scale, spacing system, and reusable components. The design emphasized readability for blog content while maintaining visual appeal for portfolio showcases. Dark glassmorphism style was chosen for a modern aesthetic.

Design Decisions:

  • Glassmorphism with subtle transparency effects
  • Typography optimized for long-form reading
  • Markdown-to-HTML rendering for blog posts
  • Lazy-loading images for performance

Stage 3: Building the Platform

The platform was built with Python and vanilla JavaScript to keep it lightweight. A custom admin panel allows creating and editing blog posts with markdown support. The portfolio section dynamically renders project cards from a SQLite database. SEO meta tags and Open Graph data are auto-generated.

Technical Highlights:

  • Custom CMS with markdown blog editor
  • SQLite database for lightweight deployment
  • Auto-generated SEO meta tags and sitemap
  • Responsive CSS with custom animations
  • Admin panel with WYSIWYG preview

Custom admin panel with markdown editor

Stage 4: Launch & Impact

The live portfolio and blog platform

The platform was deployed and shared within the developer community. Blog posts covering web development topics attracted consistent traffic. The project demonstrates creative problem-solving, end-to-end development skills, and the ability to deliver a polished product independently.

Results & Impact:

  • Successfully serves as a personal brand platform
  • 10+ blog posts published with growing readership
  • Featured in university project showcase
  • Open-sourced on GitHub with community contributions
